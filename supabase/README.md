 # Supabase

## How to use on local?

1. Run Docker Desktop
2. Start Supabase on local
    ```sh
    npx supabase@latest start
    ```
3. Open Supabase Studio
    ```sh
    open http://127.0.0.1:54323/project/default
    ```

## Tips

```sh
# Stop local DB
npx supabase@latest stop

# Reset local DB
npx supabase@latest db reset
```
